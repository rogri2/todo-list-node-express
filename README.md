# Todo list en node y express

Proyecto de lista de tareas hecha con HTML, CSS, Javascript, Node.js y Express
## Instrucciones para ejecución del proyecto:

1. Clonar proyecto.
2. Abrir terminal en carpeta raíz del proyecto.
3. Correr comando `npm install`.
4. Una vez instalado los paquetes, correr comando `npm start`
5. El proyecto debería estar corriendo correctamente en: [local host](http://localhost:3000).
